# pdf_signer

## Frontend

A React + Vite app to upload PDFs, mock-sign the last page, view, and download.

```bash
cd frontend
npm install
npm start
```

## Backend

- Node.js + Express
- Mock signing server: backend/server.js
- Listens on `http://localhost:4000`

```bash
cd mock-server
npm install
node index.js
```
